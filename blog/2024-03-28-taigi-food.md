# 菜市仔的台語 - 豬肉、雞肉、魚仔、青菜水果

## 豬肉 ê 台語

- **五花肉、腹脅肉、三層肉** sam-tsân-bah
- **喙邊肉、臉頰肉、嘴邊肉** tshuì-pinn-bah
- **老鼠肉** niáu-tshí-bah
- **腰子、腎臟** io-tsí
- **豬舌** ti-tsi̍h
- **豬跤、豬腳** ti-kha

## 雞肉 ê 台語

- **火雞** hué-ke
- **火雞母** hué-ke-bó
- **肉雞仔** bah-ke-á
- **雞公** ke-kang
- **雞尾脽** ke-bué-tsui
- **雞胸** ke-hing

## 魚仔 ê 台語

### 常見海鮮

- **七星斑魚、紅條** âng-tiâu
- **刺鯧、肉魚仔、肉鯽仔** bah-hî-á / bah-tsit-á
- **劍旗魚、劍魚** kiàm-hî
- **白帶魚、白魚** pe̍h-tuà-hî
- **白鯧** pe̍h-tshiun
- **虱目魚、海草魚** sat-ba̍k-hî / hái-tsháu-hî
- **蝦、蝦仔、蝦米、蝦仁** hê / hê-á / hê-bí / hê-jîn
- **透抽** thàu-thiu
- **小卷、小管仔** sió-kńg-á

### 常見淡水魚

- **吳郭魚、南洋鮐仔** ngôo-kueh-hî / lām-mī-tâi-á
- **草魚** tsháu-hî
- **鱸魚** lôo-hî
- **鰱魚** liân-hî

### 其他海產

- **水母、海蜇、石鏡** tsuí-bó / hái-thē / tsio̍h-kiànn
- **紅蟳** âng-tsîm
- **龍蝦** lîng-hê / liông-hê
- **鮑魚** pau-hî
- **魚翅、鯊魚翅** soa-hî-chhì

## 青菜水果正著時

遮只有標示正著時 ê 青菜水果，因為栽培技術 ê 進步，誠濟佇其他 ê 時節嘛食有喔。

|:------|:------|:------|
| 番麥 huan-be̍h | 玉米 | 7-9月 |
| 烏骨仔 oo-kut-á | 秋葵 | 4-9月 |
| 苦瓜 khóo-kue | 苦瓜 | 6-8月 |
| 茄子 kiô-á | 茄子 | 5-11月 |
| 菠薐仔 pue-lîng-á | 菠菜 | 1-3月、11-12月 |
| 莧菜 hīng-tshài | 莧菜 | 6-10月 |
| 花菜 hue-tshài | 花椰菜 | 1-4月、11-12月 |
| 蕹菜 ìng-tshài | 空心菜 | 3-11月 |
| 油菜 iû-tshài | 油菜 | 1-3月、11-12月 |

### 弓蕉 king-tsio（香蕉）

台灣的香蕉品種真濟，主要產季佇 4-10月：

- **舊北蕉** kū-pak-tsio
- **寶島蕉、新北蕉** pó-tó-tsio、sin-pak-tsio
- **玉山蕉** Gio̍k-san-tsio
- **山蕉** suann-tsio
- **粉蕉** hún-tsio
- **蛋蕉** nn̄g-tsio

### 豆仔 tāu-á

|:------|:------|:------|
| 菜豆 tshài-tāu | 豇豆 | 4-9月 |
| 敏豆 bín-tāu | 四季豆 | 全年 |
| 肉豆 bah-tāu | 扁豆 | 2-4月 |
| 荷蘭豆 huê-liân-tāu | 荷蘭豆 | 1-3月、12月 |
